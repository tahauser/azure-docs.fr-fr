# <a name="azure-analysis-services-samples"></a>Exemples Azure Analysis Services
Utilisez les exemples de ressources suivants pour découvrir et tester Analysis Services pour votre environnement.

## <a name="code-samples"></a>Exemples de code
Le dépôt [Analysis Services](https://github.com/Microsoft/Analysis-Services) sur GitHub inclut des exemples de code open source et des projets de la communauté. 

## <a name="tabular-model-project-and-database-samples"></a>Exemples de projet et de base de données de modèle tabulaire
[Adventure Works pour Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) sur GitHub est l’exemple de projet de modèle tabulaire le plus fréquemment utilisé. Vous pouvez télécharger un projet Visual Studio ou un exemple complet de base de données de modèle tabulaire.


## <a name="sample-databases-on-github"></a>Exemples de bases de données sur GitHub
Ces exemples de bases de données sur GitHub peuvent être utilisés pour créer et tester vos propres modèles. 

[Exemples de bases de données Adventure Works](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2014)

[Exemples de bases de données Wide World Importers](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Modèle d’exemple Adventure Works Internet Sales dans le portail Azure
Si vous avez un serveur Analysis Services dans le portail Azure, vous pouvez rapidement et facilement créer un exemple de modèle. Dans la page de vue d’ensemble de votre serveur dans le portail Azure, cliquez sur **Nouveau modèle** puis, dans **Choisir une source de données**, sélectionnez **Exemples de données**.
