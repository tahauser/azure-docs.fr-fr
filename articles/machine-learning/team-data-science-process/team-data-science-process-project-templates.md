# <a name="team-data-science-process-project-planning"></a>Planification des projets Team Data Science Process

Le processus TDSP (Team Data Science Process) fournit un cycle de vie qui structure le développement des projets de sciences des données. Cet article fournit des liens vers les modèles Microsoft Project et Excel qui vous aident à planifier et à gérer ces étapes de projet.

Le cycle de vie expose les principales phases que les projets exécutent généralement, souvent de manière itérative :

- Présentation de l’entreprise
- Acquisition de données et compréhension
- Modélisation
- Déploiement
- Acceptation du client

Pour obtenir une description de chacune de ces étapes, consultez [Cycle de vie du processus TDSP (Team Data Science Process)](https://docs.microsoft.com/azure/machine-learning/team-data-science-process/lifecycle).

 
## <a name="microsoft-project-template"></a>Modèle Microsoft Project

Le modèle Microsoft Project pour Team Data Science Process est disponible ici : [Modèle Microsoft Project](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.mpp) 

Lorsque vous ouvrez le plan, cliquez sur le lien à l’extrême gauche pour le TDSP. Modifiez le nom et la description, puis ajoutez d’autres ressources d’équipe dont vous avez besoin. Estimez les dates requises en fonction de votre expérience.

![1](./media/team-data-science-process-project-templates/ms-project-templates.png)

Chaque tâche possède une note. Ouvrez ces tâches pour voir quelles ressources ont déjà été créées pour vous.

![2](./media/team-data-science-process-project-templates/ms-project-template-task.png)


## <a name="excel-template"></a>Modèle Excel

Si n’avez pas accès à Microsoft Project, une feuille de calcul Excel avec les mêmes données est également disponible en téléchargement ici : [Modèle Excel](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.xlsx) Vous pouvez l’extraire pour l’utiliser dans votre outil préféré.

Utilisez ces modèles à vos propres risques. Les [clauses d’exclusion de responsabilité habituelles](https://www.gnu.org/licenses/gpl-3.0.en.html) s’appliquent.
